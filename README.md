## Sobre
Projeto desenvolvido na disciplina de IIoT, ministrada pela professora [Yuri Komuta](https://www.linkedin.com/in/yurikomuta/), com o objetivo de implementar um sensor ultrassônico.

## Descrição
A atividade consistiu no desenvolvimento de um sensor ultrassônico, onde o led é acendido quando há uma presença próxima e desligado em sua ausência.

Além disso, o terminal exibe uma mensagem indicando uma ameaça ou segurança.
